Where your knowledge lives. Must be an existing folder.

Vault backup

Local-only by default. Optionally connect the vault to a private git remote you own — to back it up, or to share it as a team knowledge base. One-time; the nightly sync never auto-pushes.

—

Set up git auth first (SSH key or OS credential manager). The URL is saved by git as the vault's remote — no token is ever stored here. Make sure the remote is private: connecting publishes your vault's contents to it.
